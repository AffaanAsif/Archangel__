## `[PROTOCOLS]` — The Three Non-Negotiables

We reject the false compromise between speed and structural safety. Today's software systems are buried under unnecessary compile-time bloat, fragile external dependencies, and poorly verified state boundaries.

```
■  BOUNDED LATENCY AND PREDICTABLE CORRECTNESS
   ──────────────────────────────────────────────────────────────
   Every execution route must maintain a deterministic latency
   budget. Clear performance bounds. GC overhead: zero tolerance.
   Allocations are budgeted, not discovered at runtime.

■  STRICT STRUCTURAL SAFETY
   ──────────────────────────────────────────────────────────────
   Static type safety. End-to-end trace modeling. Runtime
   invariants. Eliminates entire classes of concurrent race
   conditions before they can manifest in production.

■  ZERO-ABSTRACTION BLOAT
   ──────────────────────────────────────────────────────────────
   Abstractions are useful only when they do not hide execution
   overhead. Slim microservices. Lightweight message passing.
   Debugging is straightforward. Scale is predictable.
```

```

typescript@5.8      End-to-end type safety
```

## `[QUICKSTART]` — Run Locally

**Prerequisites:** Node.js 18+

```bash
# 1. Clone the repository
git clone https://github.com/AffaanAsif/Archangel__.git
cd Archangel__

# 2. Install dependencies
npm install

# 3. Configure environment
# Create .env.local with the variables below
touch .env.local

# 4. Launch development server
npm run dev
# → http://localhost:3000
```

**Production build:**

```bash
npm run build    # Compiles React (Vite) + bundles Express server
npm start        # Serves from dist/
```

## `[ENV_CONFIG]` — Environment Variables

```bash
# .env.local

GEMINI_API_KEY=your_gemini_api_key      # Required — powers AI features
SMTP_USER=your_email@gmail.com          # Optional — contact form delivery
SMTP_PASS=your_gmail_app_password       # Optional — Gmail app password
```

> Without `SMTP_USER` / `SMTP_PASS`, contact form submissions are logged server-side but not delivered via email. The application runs fully without them.
